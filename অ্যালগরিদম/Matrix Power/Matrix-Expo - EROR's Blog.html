Matrix Exponential (ম্যাট্রিক্স এক্সপো) এবং এর Application


জিনিসটা শিখছি DIU_Primtive এর সামিউল ভাইয়ের কাছে। অনেক মজার এবং কাজের জিনিস। সাহস করে কিছু একটা লিখে ফেললাম, ভুল ত্রুটি মার্জনিয়। আমার কোন কথায় ভুল আছে মনে করলে একটু কষ্ট করে আমাকে একটা মেইল করে দিবেন : sadlil.rhythm@gmail.com এই ঠিকানায়।

ম্যাট্রিক্স এক্সপো যার সহজ বাংলায় মানে হল ম্যাট্রিক্স এর পাওয়ার। A যদি একটা ম্যাট্রিক্স হয় তাহলে এর n তম এক্সপো হচ্ছে An = A*A*A*…*A। ম্যাট্রিক্সএর রো এবং কলাম এর Property ঠিক রেখে একটা ম্যাট্রিক্সকে গুন করব। এই, এতে লাভ কি?????

এতে লাভ হল এর সাহায্যে বড় বড় লিনিয়ার রিকারেন্স সমাধান করা যায়। এখন প্রশ্ন হল লিনিয়ার রিকারেন্স কি?? আমরা সবাই তো ফিবোনাক্কি সিরিজ জিনিসটার সাথে পরিচিত। ফিবোনাক্কি সিরিজ একটা লিনিয়ার রিকারেন্স। মানে হল যদি কোন ধারার n তম পদ তার পূর্ববর্তি কিছু পদের যোগফল এর উপর নির্ভর করে তাকেই সোজা কথায় লিনিয়ার রিকারেন্স বলে।
যেমন আমাদের ফিবোনাক্কি ধারা তে তার nতম পদ F(n) = F(n-1) + F(n-2);

এখন বড় বড় প্রোগ্রামাররা বলবেন এহ, আইছে। ফিবোনাক্কি তো আমরা কত্ত ভাবে সলভ করতে পারি। যেমন : ১) গুইনা গুইনা মানে ব্রুট ফোর্স আর কি, ২) ডিপি।

হুম ফিবোনাক্কি অনেক ভাবেই সলভ করা যায় তাইলে ম্যাট্রিক্স এক্সপো কেন। প্রশ্নের উত্তরঃ বললাম আমাকে 10^10 তম ফিবোনাক্কি নাম্বারটা বের করে দেন। অবশ্যই মডুলার ভ্যালু। কারন এত বড় নাম্বার কোথাও তো স্টোর করা যাবে না। তারমানে আপনি আমাকে F(10^10)%MOD এই মান টা বের করে দিন। কি করবেন?? ব্রুট ফোর্স ট্রাই করলে কয়দিন লাগে সল্যুশন আসতে আল্লাহ জানে। তাহলে কি ডিপি??? ডিপির বেসিক একটা জিনিস হল মেমোরাইজ করা, তাই না। আর ম্যামোরাইজ করতে লাগে অ্যারে। এখন 10^10 এত্ত বড় সাইজ অ্যারে কই পাবেন???? তার মানে সব ফেইল তাই না। না সব ফেইল না। কিছু পদ্ধতি তো আছেই। তার মধ্যে একটা এই ম্যাট্রিক্স এক্সপো। F(10^10)%MOD এর ভ্যালু আপনাকে বের করে দিবে ১ সেকেন্ড এর ও অনেক অনেক কম সময়ের মধ্যে।

আসেন তাহলে দেখা যাক কিভাবে ম্যাট্রিক্স এক্সপো দিয়া ফিবোনাক্কি নাম্বার সলভ করা যায়।

প্রথমে, আমরা জানি

F(1)=1;
F(2)=1;
F(3) = F(2) + F(1) = 1+1 = 2;
F(n) = F(n-1) + F(n-2);



এখন যদি আমরা ধরে নেই F(1) এবং F(2) নিয়ে একটা ২X১ ম্যাট্রিক্স A । এবং F(3), F(2) একটা ম্যাট্রিক্স B. মানে ঠিক এই রকম।



এখন আমরা A ম্যাট্রিক্স থেকে B ম্যাট্রিক্স বানানোর চেস্টা করব। কিভাবে??? কিছু একটা গুন করে। গুন করে কেন এর জবাব টা একটু পরে দিচ্ছি।
 

কিছু একটা জিনিসটা ও হবে একটা ম্যাট্রিক্স, ধরি এর নাম Base Matrix (BM). আবার ধরি BM দেখতে নিচের ছবিটার মত। এটা একটা ২x২ স্কয়ার ম্যাট্রিক্স। (সব শুধু ধরতেছিই, ভাওতামি?? চিন্তা নাই একটু পরে সব ক্লিয়ার)


কেন আমরা গুন করতেছি এবং কেন BM টা কে এই রকম করে ধরছি।

আমরা জানি F(3) = F(2) + F(1) তাই না? এখন নিচের ছবিটার দিকে একটু তাকাই।


সুন্দর ভাবে দেখা যাচ্ছে Matrix A এর সাথে যদি আমরা Matrix BM গুন করি তাহলে আমরা Matrix B পাচ্ছি। কারনটা কি, কারনটা সহজ ম্যাট্রিক্স গুনের পদ্ধতি। যদি ভুলে গিয়ে থাকি তাহলে একটু এইটা দেখা যায়। ম্যাট্রিক্স সব সময় প্রথমটার রো এবং দ্বিতিয়টার কলাম গুন করে যোগ করতে থাকে এবং এর ফলে আমরা আমাদের লিনিয়ার রিকারেন্সটা পেয়ে যাই।

এখন যদি আমরা আর একটু সামনে আগাই তাহলে কি পাই। আমরা এখন এইটা জানি F(4) = F(3) + F(2). চার নাম্বার ছবিটা থেকে আমরা পাই এইটা।


আবার আমরা Already জানি


তাহলে Picture 6 এবং Picture 5 থেকে আমরা পাই



এখনা দেখা যাচ্ছে আমরা যদি Base Matrix BM এর স্কয়ার এর সাথে আমাদের A Matrix গুন করি গুনফল হিসেবে আমরা F(4) এবং F(3) এর একটা ম্যাট্রিক্স পাব। একই রকম ভাবে যদি F(5) এবং F(4) বের করতে চাই তাহলে কি হবে।
এখনে দেখা যাচ্ছে আমরা F(5) এবং F(4) বের করে ফেলেছি মাঝখানের F(3) ছারাই।

তাহলে জেনারেল ফরমুলাটা কি হবে???? 

মানে এখনে n এর মান 10000000000 হলে এই ফরমুলা দিয়ে আমরা 10000000000তম ফিবোনাক্কি সংখ্যা ( অবশ্যই মডুলার ভ্যালু) বের করে ফেলতে পারব মাঝখানের সব ফিবোনাক্কি নাম্বার ছাড়াই।

এখন এই প্রসেস এর সব চেয়ে কঠিন কাজ হচ্ছে Base Matrix টা Define করা। ফিবোনাক্কির জন্যতো বলেই দিলাম। যদি রিকারেন্স এই রকম হয়
 F(n) = a*F(n-1) + b*F(n-2) তাহলে আমাদের Base Matrix টা হবে এই রকম। 
সহজ, তাই না। আর ভাল ভাবে জানতে জোবায়ের ভাইয়ের এই লেখাটা পড়া যায়। 



কোডিং পার্টঃ
অনেক তো চিল্লা পাল্লা করলাম এখন কিছু কোড করার পালা। এই চিল্লা পাল্লা যা যা বললাম এই গুলা কোড কেমনে করব। আমি অনেক বড় করে কোড করছি। যাতে জিনিস টা আমার কাছে সহজ থাকে। আর আমি class Use করছি আপনি চাইলে তা struct দিয়েও করতে পারবেন। 
long long mod; // যেই ভ্যালুটা দিয়ে মড করব।

class Matrix 
{
public:

    int row, col; // Matirx এর row এবং column এর সংখ্যা
    long long valu[5][5]; // Matirx এর ভ্যালু


    void clear()  // Matirx এর সকল ভ্যালু 0 করে দেয়ার জন্য
    {
        memset(valu,0,sizeof(valu));
        row=0;
        col=0;
    }

 

    void identify(int x) // Identity Matrix বা 1 Matrix বানানোর জন্য।
    {
        clear();
        row=x;
        col=x;

        for(int i=0; i<row; i++)
            valu[i][i]=1; // ডায়াগোনাল সব এলিমেন্ট 1 করে দেয়া হচ্ছে।
    }

 

    Matrix operator * (Matrix b) // দুইটা  Matrix গুন করে গুনফল Matrix টা রিটার্ন করবে
    {
        int i,j,k;
        Matrix c;

        c.clear();
        c.row=row;
        c.col=b.col;
                               // Complexity d^3. যদিও এইটা বেশি হবে না কারন আমার    
                               // Matrix এর সাইজ অনেক ছোট। ২x২ মাত্র।

        for(i=0; i<row; i++)
            for(j=0; j<b.col; j++)
                for(k=0; k<col; k++)
                {
                    c.valu[i][j] = c.valu[i][j] + (valu[i][k])*(b.valu[k][j]);
                    c.valu[i][j] %= mod;
                }
        return c;
    }

 

    Matrix operator ^ (long long p)   // Fast Exponential for Matrix, Matrix^p রিটার্ন করে।
    {
        Matrix res, x;
        res.identify(row);

        x=*(this);

        while(p)
        {
            if(p & 1) res = res * x;
            x=x*x;
            p>>=1;
        }
        return res;

    }

};

Matrix a, b;

void setMatrixValues() //Base Matrix এবং Matrix A এর ভ্যালু সেট করতেছি।

{
    a.row=2;
    b.row=2;
    a.col=1;
    b.col=2;

    b.valu[0][0]=1;
    b.valu[0][1]=1;
    b.valu[1][0]=1;
    b.valu[1][1]=0;

    a.valu[0][0]=1;
    a.valu[1][0]=1;

}

 

int main()

{

    Matrix x,y;

    setMatrixValues();

     mod=10000000+7;
     y = b^(10000000000-2); // Base Matrix এর পাওয়ার করতেছি।
     x= y * a;              // Exponent Matrix এর সাথে A Matrix গুন দিলাম।
     printf("%lld\n",x.valu[0][0]); // Printing The Result

    return 0;
}
